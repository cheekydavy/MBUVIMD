## ⚙️ Config Vars

- `SESSION_ID`: from step 2
- `OWNER_NUMBER`: e.g., `2547XXXXXXXX`
- `PREFIX`: default `.`
- `ALWAYS_ONLINE`: `true` or `false`
- `AUTO_LIKE_STATUS`: `true` or `false`
